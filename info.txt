- Servers created with Beza Cloud may take up to 10 minutes to be able to access
- Servers created with Beza Cloud can only be opened through Beza Cloud for
  the first time on the device they were created on. After it has been connected to
  once, it will be available for connection on all your devices with Beza Cloud.
- Beza Cloud may not be able to connect to certain servers if they were created
  outside of Beza Cloud. It may be necessary for you to manually input the password
  a single time, and then Beza Cloud will be able to connect to it any time after.

npm run dist
